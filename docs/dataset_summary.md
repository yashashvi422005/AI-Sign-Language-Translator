# Dataset Summary

## Phase 2 Raw Dataset

Raw webcam videos were collected locally for the first vocabulary set.

The raw `.mp4` files are intentionally ignored by Git because video datasets can become large. Git tracks only the folder structure using `.gitkeep` files.

## Notes

- Each video is approximately 3 seconds.
- Multiple natural gesture variations can be stored under the same label when they mean the same English word.
- The folder name is the training label used in later phases.
