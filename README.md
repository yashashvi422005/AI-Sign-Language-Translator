# AI-Powered Real-Time Sign Language Translator

Professional portfolio project for real-time sign language detection, translation, and speech output.

## Current Phase

Phase 0: Environment Setup

## Project Structure

- `dataset/` - raw and collected sign language data
- `notebooks/` - experiments, visualizations, and learning notebooks
- `models/` - trained model files
- `src/detection/` - webcam and hand detection code
- `src/training/` - model training scripts
- `src/inference/` - real-time prediction scripts
- `src/utils/` - shared helper functions
- `webapp/` - web application files
- `docs/` - documentation and notes
- `tests/` - automated tests
